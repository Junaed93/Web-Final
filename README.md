## üöÄ How to Run

Follow these steps to set up and run the projects locally:

### 1. Move Files to XAMPP

Copy the entire folder into your XAMPP's `htdocs` directory:

```bash
# Example path
C:\xampp\htdocs\Web-Final
```

### 2. Start Services

Open the **XAMPP Control Panel** and start:
* Apache
* MySQL

### 3. Database Setup (If applicable)

For projects requiring a database (e.g., in folder `252/ques3`):
1. Open your browser and go to `http://localhost/phpmyadmin`.
2. Create a new database named as specified in the `.sql` file (e.g., `sundarban`).
3. Select the database and use the **Import** tab to upload the `.sql` file found in the project directory.

### 4. Access the Projects

Navigate to the projects in your web browser:
* `http://localhost/Web-Final/243/`
* `http://localhost/Web-Final/251/`
* `http://localhost/Web-Final/252/`
